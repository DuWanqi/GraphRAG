# 白名单约束机制改进

## 改进目标

确保生成内容不添加 RAG 未提供的实体和细节，同时允许模型改写表述使其自然融入叙事。

## 核心原则

✅ **允许**：改写表述、调整语序、融入叙事  
❌ **禁止**：添加未提供的实体、细节、数据、政策名称

## 改进内容

### 修改的文件
- `src/generation/novel_content_extractor.py` - `NovelContentBrief.format_for_prompt()` 方法

### 改进前后对比

**改进前**：
```
新增实体（原文未提及）：
- 改革开放: 1978年十一届三中全会确立了改革开放政策
- 深圳特区: 深圳是中国最早设立的经济特区之一
```

**改进后**：
```
可用的新知识（可改写表述，但不可添加未提供的实体或细节）：

1. [改革开放]
   1978年十一届三中全会确立了改革开放政策
2. [深圳特区]
   深圳是中国最早设立的经济特区之一

使用规则：
✓ 可以改写上述内容的表述方式，调整语序，使其自然融入叙事
✓ 可以选择性使用（不必全部使用），选择与叙事最相关的 1-3 条
✗ 不可添加上述列表中未提及的实体、人名、地名、机构名
✗ 不可添加上述内容中未提及的具体数据、政策名称、时间节点
✗ 不可推断上述内容中未提及的因果关系、影响或评价
```

## 使用示例

### ✅ 合规示例

**输入**：`1980年，我考上了大学。`

**RAG 提供**：`改革开放: 1978年十一届三中全会确立了改革开放政策`

**生成结果**：
```
1980年，我收到了大学录取通知书。那是改革开放刚刚开始的年代，
两年前的十一届三中全会确立了新政策，整个国家都在发生变化。
```

**分析**：
- ✓ 改写了"1978年" → "两年前"
- ✓ 改写了"改革开放政策" → "新政策"
- ✓ 没有添加未提供的实体

### ❌ 违规示例

**生成结果**：
```
1980年，我收到了大学录取通知书。那是改革开放后的第二年，
包产到户政策让农民有了自主权，父亲说邓小平是个伟人...
         ^^^^^^^^                    ^^^
         未提供的政策                未提供的人名
```

**问题**：
- ✗ 添加了"包产到户"（RAG 未提供）
- ✗ 添加了"邓小平"（RAG 未提供）

## 验证机制

系统会通过 `novel_content_grounding_metric` 检查生成内容：

```python
grounding = novel_content_grounding_metric(memoir_text, generated_text, novel_brief)
# grounding.value: 0.0-1.0（1.0 表示所有新内容都有依据）
# grounding.explanation: "新事实 5 个，有依据 3 个，无依据 2 个"
```

如果模型添加了未提供的实体，grounding 分数会降低。
